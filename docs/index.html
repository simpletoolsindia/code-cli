FAQ & Troubleshooting

Common questions and solutions

"beast: command not found" after install

Add the npm global bin to your PATH:

export PATH="$HOME/.npm-global/bin:$PATH"

Add this to your ~/.bashrc or ~/.zshrc

How to use local models (Ollama)?

1. Install Ollama: brew install ollama

2. Start Ollama: ollama serve

3. Pull a model: ollama pull llama3.2

4. Run Beast: beast --provider ollama --model llama3.2

Context window limit errors

Beast CLI automatically compacts context at 95% usage. To manually trigger:

beast --compact

Or type /compact in the REPL.

How to change themes?

Available themes: catppuccin-mocha, dracula, nord, tokyonight, gruvbox, monokai, one-dark

beast --theme catppuccin-mocha

Or set in config: theme: dracula

Permission denied for file operations

Beast has a permission system. Safe commands auto-approve. For new operations:

Type y to allow or a for session-wide approval.

To bypass: BEAST_MODE=bypass beast
